# Phase 1 pre-registration

`PREREGISTRATION_phase1.md` is the frozen pre-registration for the
longitudinal-utility experiment. It is preserved as a provenance artifact,
including its original date, commit hash, terminology, and table references.

The executable fine-tuning and spine-signal implementation used by the release
is `../finetune/finetune_cifar10_spine_smoke.py`. Earlier step-by-step scaffold
files contained unresolved experimental hooks and are intentionally not part of
the runnable release. Checkpoints, cached spines, logs, and raw Phase 1 results
are also excluded, consistently with `RELEASE_MANIFEST.md`.
